CHALLENGE - CONCURSO DE PREGUNTAS Y RESPUESTAS

Condiciones para uso: 
1. Debe tener xampp instalado, en caso de no tenerlo puede descargarlo aqui: https://www.apachefriends.org/es/index.html
2. Usar preferiblemente IDE de Netbeans

Pasos para uso:
1. Importar nuestra base de datos el cual esta en el paquete "bd" de nuestro proyecto.
2. Importar en la libreria de nuestro proyecto el conector mysql que se encuentra en el paquete "conector" de nuestro proyecto.
