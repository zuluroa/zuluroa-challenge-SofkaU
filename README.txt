CHALLENGE - CONCURSO DE PREGUNTAS Y RESPUESTAS

Condiciones para uso: 
1. Debe tener xampp instalado, en caso de no tenerlo puede descargarlo aqui: https://www.apachefriends.org/es/index.html
2. Usar preferiblemente IDE de Netbeans.
3. Contar con JDK 8U111 o 8U202

Pasos para uso:
1. Clonar el proyecto con el siguiente enlace https://github.com/zuluroa/zuluroa-challenge-SofkaU.git .
2. Activar los servicios de Apache y MySQL y XAMPP.
3. Dirigirnos a la pantalla principal de phpMyAdmin y crear una nueva base de datos.
4. Importar nuestra base de datos el cual esta en el paquete "bd" de nuestro proyecto.
5. Importar en la libreria de nuestro proyecto el conector mysql que se encuentra en el paquete "conector" de nuestro proyecto.
